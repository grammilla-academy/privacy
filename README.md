# Grammilla Privacy Site

Files for the public support/privacy page:

- `index.html` - support landing page with email-only contact
- `privacy.html` - public privacy policy page

Recommended GitHub setup:

1. Create GitHub Organization: `grammilla-academy`
2. Create public repository: `privacy`
3. Upload these files to the repository root
4. In GitHub: `Settings` -> `Pages`
5. Set:
   - `Source`: `Deploy from a branch`
   - `Branch`: `main`
   - `Folder`: `/ (root)`
6. Wait for GitHub Pages to publish

Expected URL:

`https://grammilla-academy.github.io/privacy/`

After publishing, update the app's privacy URL to:

`https://grammilla-academy.github.io/privacy/privacy.html`
